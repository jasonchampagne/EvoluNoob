## Demander sans cesse à l'utilisateur de saisir une lettre tant que celle-ci n'est pas une voyelle

```python
is_vowel = False

while not is_vowel:
    user_letter = input("Saisir une voyelle : ")
    
    if user_letter in "aeiouy":
        is_vowel = True
        print("Correct !")
```

## Parcourir une liste et afficher uniquement les valeurs négatives

```python
numbers = [16, -8, 0, 1, -325, 128.45, -1, -36.9, 97, 20]

for n in numbers:
    if n < 0:
        print(n)
```
