## Convertir une température de Celcius à Fahrenheit

La formule connue pour convertir des degrés Celcius en degrés Fahrenheit est `F° = C° x 1,8 + 32`. Il suffit alors d'écrire le calcul et affecter le résultat à notre variable `farenheit_temperature` :

```python
celcius_temperature = 31
farenheit_temperature = celcius_temperature * 1.8 + 32

print(celcius_temperature, "°C =", farenheit_temperature, "°F")
```

## Calculer le périmètre d'un cercle

Une formule connue pour calculer le périmètre d'un cercle est `2 x π x rayon`. Il suffit alors d'écrire le calcul et affecter le résultat à notre variable `circle_perimeter` :

```python
PI = 3.14
radius = 15
circle_perimeter = 2 * PI * radius

print("Périmètre d'un cercle de rayon", radius, ":", circle_perimeter)
```
