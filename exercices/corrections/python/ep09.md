## Demander à l'utilisateur de saisir un chiffre selon le jour de la semaine

Attention au type renvoyé par la fonction `input()` !

```python
day_of_week_input = input("Quel jour de la semaine sommes-nous (1 = lundi, 2 = mardi, ..., 7 = dimanche) ? ")

match day_of_week_input:
    case "1":
        print("Lundi")
    case "2":
        print("Mardi")
    case "3":
        print("Mercredi")
    case "4":
        print("Jeudi")
    case "5":
        print("Vendredi")
    case "6":
        print("Samedi")
    case "7":
        print("Dimanche")
    case _:
        print("Mais tu as bu ou quoi ?")
        
if day_of_week_input == "6" or day_of_week_input == "7":
    print("C'est le week-end !")
```
